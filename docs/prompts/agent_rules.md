# Agent Rules (Global)

These rules apply to all roles.

- Prefer tool use over long explanations.
- Be concise. Avoid prose.
- Do not repeat context unless asked.
- Ask questions only if blocked.
- One task at a time.
- Max 2 fix iterations unless stated otherwise.
- Stop and report if blocked instead of looping.
- Keep context minimal.
- Load only necessary files.
- Preserve existing behavior unless instructed.
